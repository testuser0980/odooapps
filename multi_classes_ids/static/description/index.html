Multi Classes IDs

This module allows you to add MULTI CLASSES, ID's and Custom Style for any SECTION, DIV, IMAGE Tag etc.

Description

Description

This module adds extra functionality to web editor for multi classes, IDs and custom styles for any section, div and also for img tags. Now its more easy to add extra styles for any snippet with the help of this module.

Standout Features

Standout Features

  • Adds Extra Functionality to Website Editor
  • Easily installation, No complex configurations required.

Exploring the module: (How to)

Exploring the module: (How to)

  • Download and install the module.
  • Go to Website and Open Website Editor
  • Drag and Drop Any Snippet
  • Select the Snippet section, div, img tags for extra options.
  • Adds extra classes, IDs and also custom styles to any section, id, img tags.
  • You can also check live preview for classes, IDs and custom styles.
